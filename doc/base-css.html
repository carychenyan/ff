表单控件描述

FORM表单组成元素，包括：输入框，文本框，标签，单选，多选，图片 psd文件。 psd文件。
单个控件是组合而成复杂表单的元素。此处列举web的标准表单控件，并补充相应的案例，以供参考，使用和拓展。（表单制作时候，也应考虑相应浏览器的兼容问题）。

DEMO： component 中 form.html

表单设计和制作应考虑的几个因素：

表单设计和制作应考虑的几个因素：
安全: 减少用户的信息泄漏，合适的考虑用户在某些情况下，提交过程小，被他人看到等类似的行为。
可靠: 在写文字，上传等需要较长时间的用户行为时，性能劲量可靠，减少因为网络等问题引发提交失败后，用户需要重新写文章等类似的情况。
快速: 用户提交后，能够快速反映并提交，无停顿或尽量减少停顿，可通过后台运行的方式。个别情况出外。
反馈: 在用户填写信息的时候，尽量能够避免和减少用户的无效提交，避免在提交结果后，告知用户用户名已经注册等类似情况

输入框

不可编辑状态

对于在表单中呈现不可编辑的数据，无需使用实际的表单控件。
